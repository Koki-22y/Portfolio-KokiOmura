1." ADS-Bデータによる航空機位置マッピングツール"
公開WebSocketからリアルタイムでADS-Bデータを取得し、航空機の位置・高度・コールサインなどを抽出するツールです。

10km以上を飛行する機体を自動検出
DataFrameに整形して保存
後で可視化や飛行パターン分析がしやすい形に

リアルタイムデータ処理とJSONハンドリングの練習になりました。

2." ひらがな手書き文字分類（CNN）"
手書きひらがなをCNNで認識するモデルです。

TensorFlow/Kerasでゼロから構築
データ拡張・学習率スケジューリング実装
混同行列と評価指標の可視化
未知の画像を予測するスクリプト付き

画像認識の基本をしっかり固めました。

3." 時系列予測モデルの説明性向上（TimeSHAP拡張）"
– トゥウェンテ大学 学士論文プロジェクト
オープンソースのTimeSHAPライブラリを改造・拡張し、LSTM/GRUの予測根拠を人間が見やすい形にしました。

可視化レイヤーを全面的に作り直し
TimeSHAPの出力を見やすく改善
ヒートマップで重要タイムステップを一目でわかるように
モデル訓練 → 説明生成 → 可視化 のフルパイプライン完成
Jupyter Notebookで再現性確保

「どの時点のどの特徴量が予測に効いたか」を直感的に伝えられるツールに仕上げました。
